# Badanie układu odzyskiwania energii z drgań UAV typu płatowiec z wykorzystaniem piezo-elementów

Celem pracy było przeprowadzenie analizy numerycznej profilu skrzydła płatowca klasy mikro w celu zaprojektowania układu odzyskiwania energii z drgań obiektu latającego z wykorzystaniem piezoelementów. W pracy przedstawiono przegląd stosowanych rozwiązań oraz podstawowe zagadnienia związane z tematyką odzyskiwania energii na podstawie dostępnej literatury oraz źródeł internetowych. Analizę numeryczną wykonano w oparciu o model skrzydła bezzałogowego statku powietrznego typu S1223  wykonanego w programie SolidWorks. W module Ansys Fluent przy wykorzystaniu pakietu CFD zasymulowano przepływ powietrza o charakterze turbulentnym wokół profilu skrzydła. Badania przepływu zostały wykonane dla różnych kątów natarcia i różnych prędkości skrzydła. Następnie wykonano analizę mechaniczną skrzydła pod kątem rozkładu naprężeń występujących na nim oraz  jego deformacji. Analiza modalna umożliwiła wyznaczenie częstotliwości drgań własnych. Na tej podstawie przeprowadzono analizę drgań harmonicznych skrzydła i wyznaczono częstotliwości rezonansowe zapewniające najbardziej efektywne działanie piezoelementu. Wyznaczenie największych wartości sił działających w poszczególnych węzłach modelu umożliwiło zlokalizowanie optymalnego położenia piezoelementu na profilu skrzydła.
